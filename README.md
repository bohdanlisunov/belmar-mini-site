# Belmar Mini Site

Mini project for Belmar API integration test.

## Features
- Add Lead page (form with firstName, lastName, phone, email)
- Lead Statuses page (table with filter by date)
- Serverless functions for API calls (`addlead` & `getstatuses`)
- Hosted on Netlify: https://gentle-sorbet-7817cf.netlify.app

## Tech Stack
- HTML, CSS, JavaScript
- Netlify Functions (serverless proxy)
- GitHub for source code

## How to use
1. Open Add Lead page (`index.html`) and submit a lead.
2. Open Lead Statuses page (`statuses.html`) to see lead statuses.
